
# Conda init and usage
When want to run or install or do other things, make sure it is in the correct conda env. 
```powershell
# Activate environment (may work with proper PATH)
conda activate a2d-studio
# Or use conda.bat
D:\aaa-new\setups\conda\conda-files\condabin\conda.bat activate a2d-studio
```
