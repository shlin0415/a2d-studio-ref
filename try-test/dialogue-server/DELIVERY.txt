## 🎯 What's Inside

### Core Components (6 Total)

1. **CharacterLoader** - Load character data
   - Read setting.txt (YAML format)
   - Parse enhanced TXT with multiline strings
   - Discover emotion figures in fig/ folders
   - Returns CharacterSettings objects

2. **PromptBuilder** - Build system prompts
   - Create prompts with dialogue format rules
   - Enforce 【emotion】text（actions）<translation> format
   - Support 18 standard emotions
   - Multi-language support (ZH, EN, JP)

3. **LLMService** - LLM API wrapper
   - OpenAI API integration
   - Streaming and non-streaming generation
   - Async/await support
   - Error handling and logging

4. **DialogueParser** - Parse LLM output
   - Extract emotion tags from 【emotion】text format
   - Get dialogue, actions, translations
   - Validate emotions against standard list
   - Multi-character assignment (round-robin)

5. **OutputGenerator** - Generate structured output
   - JSONL format (primary, like SillyTavern)
   - JSON, CSV, TXT alternatives
   - Metadata headers with character info
   - File I/O support

6. **DialogueGenerator** - Main orchestrator
   - Coordinate all 5 components
   - generate_multi_character_dialogue() method
   - generate_single_character_dialogue() method
   - Full async pipeline

### Data Models

- CharacterSettings - Character configuration
- DialogueLine - Single dialogue unit
- DialoguePlaybook - Complete playbook
- STANDARD_EMOTIONS - 18 emotions list

## 🚀 Quick Start

### 1. Install Dependencies
```bash
cd dialogue-server
pip install -r requirements.txt
```

### 2. Run Interactive Demo
```bash
python quick_start.py
```

### 3. Run Test Suite
```bash
python tests/test_components.py
```

### 4. Use in Your Code
```python
import asyncio
from dialogue_gen.dialogue_generator import DialogueGenerator

async def main():
    gen = DialogueGenerator()
    result = await gen.generate_multi_character_dialogue(
        character_paths=[
            "Character-fig-setting-example/ema",
            "Character-fig-setting-example/hiro",
        ],
        topic="Two friends meet at a bookstore",
        output_path="output.jsonl",
    )
    print(f"Generated {result['num_lines']} lines")

asyncio.run(main())
```
