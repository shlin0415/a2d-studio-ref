艾玛: 【开心】希罗，今天也辛苦啦～（从背后轻轻抱住正在整理床铺的希罗，把脸贴在她背上）
  → ヒロ、今日もお疲れ様～

希罗: 【害羞】别、别突然抱上来啊...（身体微微一僵，但并没有挣脱）
  → いきなり抱きつかないでよ…

艾玛: 【有点开心】因为希罗的后背很温暖嘛。而且...今天一整天都没怎么抱到，我有点寂寞了哦？（蹭了蹭）
  → だってヒロの背中は温かいんだもん。それに…今日は一日ほとんど抱っこできなかったから、ちょっと寂しかったの？

希罗: 【慌张】说、说什么呢...该睡觉了，快去换睡衣。（耳朵微微发红，转过身来）
  → な、何言ってるの…もう寝る時間よ、早くパジャマに着替えて。

艾玛: 【平静的思考】嗯...那希罗帮我换？就像上次那样？（歪着头，露出狡黠的笑容）
  → うん…じゃあヒロが着替えさせてくれる？（首をかしげて、いたずらっぽい笑みを浮かべる）前みたいに？

希罗: 【震惊】上次那是...！那是你假装手受伤了才...！（脸瞬间涨红）
  → 前回は…！（顔が一気に赤くなる）あれはあなたが手を怪我したふりをしたから…！

艾玛: 【害羞】但是希罗很温柔地帮我换了呀。今晚也...不行吗？（向前一步，仰视着希罗）
  → でもヒロは優しく着替えさせてくれたよね。（一歩前に出て、ヒロを見上げる）今夜も…ダメ？

希罗: 【无奈】...真拿你没办法。只此一次，下不为例。（叹了口气，伸手轻轻戳了戳艾玛的额头）
  → …本当にしょうがない子ね。（ため息をつき、指でそっとエマの額をつつく）今回だけよ、次はダメだから。

艾玛: 【开心】嗯！最喜欢希罗了～（开心地举起双手，配合着让希罗帮她脱下外套）
  → うん！ヒロ大好き～（嬉しそうに両手を上げ、ヒロが上着を脱がせるのに合わせる）
