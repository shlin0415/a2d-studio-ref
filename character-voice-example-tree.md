~/ref/Character-voice-example
It contains examples of folders that include weights, character voice files and settings.
```
.
├── 希罗
│   └── GPT-SoVITS
│       ├── good_ref
│       │   ├── 0204Adv10_Hiro006.ogg
│       │   ├── 0204Adv10_Hiro006.txt
│       │   ├── 0205Trial09_Hiro093.ogg
│       │   └── 0205Trial09_Hiro093.txt
│       ├── hiro-e15.ckpt
│       ├── hiro_e8_s2184.pth
│       ├── other_ref
│       └── voice_setting.txt
└── 艾玛
    └── GPT-SoVITS
        ├── good_ref
        │   ├── 0101Adv26_Ema012.txt
        │   └── 0101Adv26_Ema012.wav
        ├── manosaba_ema-e15.ckpt
        ├── manosaba_ema_e8_s864.pth
        ├── other_ref
        └── voice_setting.txt
```
By default, the tool should be able to load the character voice in such format.
