ema: 【平静】今天的学生会工作又拖到这么晚呢，hiro。（侧躺在枕头上，手指卷着发梢）
  → 今日の生徒会仕事もまたこんなに遅くまでかかったね、hiro。

  → 仕方ない、卒業シーズンの書類整理はいつも特に煩雑なんだ。

ema: 【有点开心】但是hiro认真工作的样子很帅气哦。（伸手拉住hiro的袖口）
  → でもhiroが真面目に働く姿はとてもかっこいいよ。

  → そんなこと突然言わないで……

ema: 【微笑默认】呐，hiro。（轻轻把hiro拉近，声音放低）
  → ねえ、hiro。

hiro: 【慌张】等、等一下，我还没换睡衣……（身体有些僵硬）
  → 待って、私はまだパジャマに着替えてない……

ema: 【平静的思考】没关系哦，反正待会儿也要脱掉的。（指尖划过hiro的衬衫纽扣）
  → 大丈夫だよ、どうせあとで脱ぐことになるんだから。

hiro: 【震惊】你、你在说什么啊！（声音不自觉地提高）
  → あなた、何を言ってるの！

ema: 【害羞】难道hiro今晚不想抱着我睡吗？（做出委屈的表情，但眼中带着狡黠）
  → まさかhiroは今夜、私を抱いて寝たくないの？

hiro: 【无奈】……真是拿你没办法。（妥协地叹了口气，开始解衬衫纽扣）
  → ……本当にあなたには手を焼くよ。

ema: 【开心】最喜欢这样的hiro了。（满意地看着hiro的动作，钻进被窝）
  → こんなhiroが一番好きだ。

hiro: 【害羞】安静点……明天早上还有课呢。（关掉台灯，在黑暗中搂住ema）
  → 静かにして……明日の朝も授業があるんだから。

ema: 【平静】hiro的心跳声，好快哦。（把耳朵贴在hiro胸前）
  → hiroの鼓動、すごく早いね。

hiro: 【慌张】那、那是因为……（把脸埋进ema的头发里）
  → それは、だって……

ema: 【微笑默认】晚安吻呢？（仰起脸，在黑暗中寻找hiro的嘴唇）
  → おやすみのキスは？

hiro: 【害羞】……真是的。（轻轻吻上ema的唇，短暂停留后分开）
  → ……もう。

ema: 【开心】不够哦。（再次凑近，声音带着笑意）
  → 足りないよ。

hiro: 【无奈】……这是最后一次了。（加深了这个吻，手臂不自觉地收紧）
  → ……これで最後だからね。

ema: 【平静】嗯，今晚就先这样吧。（满足地窝在hiro怀里）
  → うん、今夜はこれくらいにしておくね。

hiro: 【默认】睡吧，我会一直在这里的。（轻抚ema的头发，声音温柔）
  → 寝よう、私はずっとここにいるから。
