# Character-fig-setting-example
~/ref/Character-fig-setting-example
It contains examples of folders that include character emotion figs and settings.
```
.
├── ema
│   ├── fig
│   │   ├── 下定决心.png
│   │   ├── 发现问题.png
│   │   ├── 困难地思考.png
│   │   ├── 害羞.png
│   │   ├── 平静.png
│   │   ├── 平静的思考.png
│   │   ├── 开心.png
│   │   ├── 微笑着害羞.png
│   │   ├── 悲伤.png
│   │   ├── 惊讶.png
│   │   ├── 慌张.png
│   │   ├── 担心.png
│   │   ├── 无奈.png
│   │   ├── 有点开心.png
│   │   ├── 有点悲伤和惊讶.png
│   │   ├── 生气.png
│   │   ├── 辩驳.png
│   │   ├── 震惊.png
│   │   └── 默认.png
│   ├── setting-calm-version.txt
│   ├── setting-straightforward-version.txt
│   ├── setting.txt
│   └── 头像.png
└── hiro
    ├── fig
    │   ├── 不知所措.png
    │   ├── 发现问题.png
    │   ├── 呆滞.png

    │   ├── 微笑默认.png
    │   ├── 思考.png
    │   ├── 悲伤.png
    │   ├── 无语.png
    │   ├── 生气.png
    │   ├── 自信.png
    │   └── 默认.png
    ├── setting-calm-version.txt
    ├── setting-straightforward-version.txt
    ├── setting.txt
    └── 头像.png
```
By default, the tool should be able to load the character in such format.
